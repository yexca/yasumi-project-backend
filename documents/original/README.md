# Original Development Documents

This directory preserves backend-local historical development records.

## What Belongs Here

- round plans
- phase-by-phase development notes
- acceptance checklists tied to a specific delivery round
- release and operational notes kept for historical traceability

## Current Archive Structure

- `round-01-mvp-backend-delivery/`
  - archived Round 01 MVP backend delivery notes
- `round-02-real-sync-and-structure/`
  - archived Round 02 backend planning, acceptance, and structure notes

## Usage Rules

- Treat this directory as archive material, not the primary guide for current backend changes.
- If an archive document differs from current code, prefer the current code and `../developer/`.
- Only make path fixes, redaction, errata, or index adjustments here unless a user explicitly asks for historical rewriting.
